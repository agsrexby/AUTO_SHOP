Содержание

Это приложение предназначено для управления данными магазина автозапчастей. С его помощью вы можете эффективно работать с клиентами, менеджерами, запчастями, заказами и другими аспектами деятельности магазина.

Для работы с приложением выполните следующие шаги:

  1. Выберите нужную категорию данных из меню (например, "Клиенты", "Менеджеры", "Запчасти").
  2. После выбора категории появятся четыре кнопки управления:
    • "Добавить" — нажмите, чтобы открыть форму добавления новой записи.
    • "Обновить" — выберите запись и измените данные с помощью открывшейся формы.
    • "Удалить" — выберите запись и удалите её из базы данных.
    • "Поиск" — используйте для поиска нужной записи по ключевым словам.
  3. Под кнопками управления отображается таблица с данными из выбранной категории.
  4. Для сортировки данных нажимайте на заголовки столбцов таблицы (например, "ID", "Название", "Цена").

Приложение предоставляет удобный интерфейс для выполнения основных операций: добавления, обновления, удаления и поиска записей. Сортировка и фильтрация данных помогают быстро находить и анализировать информацию.
